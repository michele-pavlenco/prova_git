Info git

  • all'inizio di ogni nuova implementazione, si fa pull e si crea un nuovo branch, a partire dal main
  • si continua a lavorare e committare sul proprio branch
  • al termine dello sviluppo, ci si sposta sul branch main e si esegue un pull
  • si esegue il merge
  • si cancella il branch creato al punto 1
  • si esegue il push
